# ResuAI - AI Resume Analyzer

An AI-powered resume analysis tool built with React, Node.js, and OpenAI.

## Features
- Resume scoring (Overall, ATS, Clarity, Impact, Tone)
- Skills detection and gap analysis
- Career path recommendations
- Resume enhancement suggestions
- Interview question generation

## Tech Stack
- **Frontend:** React + TypeScript + Vite + Tailwind CSS
- **Backend:** Node.js + Express + TypeScript
- **Database:** PostgreSQL (via Drizzle ORM)
- **AI:** OpenAI GPT-4o

## Setup & Deploy

See deployment instructions for Railway.app in the docs.

### Environment Variables Required
```
PORT=3000
DATABASE_URL=postgresql://...
SESSION_SECRET=your-secret-key
```
